# Instagram カルーセル投稿、ベータテスター募集

## 用途
1968 ベータテスター募集の Instagram カルーセル投稿（スワイプ式 6〜8 枚）。
Canva / Figma / Keynote などで作成、最終的にインスタグラム投稿として配信する。

## 投稿先候補
- 1968 公式アカウント（要開設）
- 油谷さん個人アカウント
- 1968 年生まれが集まる関連コミュニティへの紹介投稿（許可を得て）

---

## ビジュアルガイドライン

### サイズ
- **正方形 1080 × 1080 px**（スワイプ全枚共通、安全圏 70 × 70 px）
- 縦長 1080 × 1350 px のオプションあり、より画面占有率が高い

### カラー（CLAUDE.md「墨・紺・生成り」基調）
- 背景、生成り `#F8F4EC`（メインベース）
- アクセント、紺 `#1E3A5F`（タイトル・ボタン）
- 文字、墨 `#1F1F1F`（本文）
- アクセント 2、土色 `#8B6F3D`（アンダーライン・タグ）
- 補助、淡い灰 `#C9BFAC`（区切り線）

**禁止色**、ピンク・蛍光色・ネオン系。

### フォント
- 見出し、Noto Sans JP Bold、または明朝系（A-OTF リュウミン Pro 等）でも可
- 本文、Noto Sans JP Regular
- 数字、Outfit / Inter など欧文サンセリフでアクセント可

### モチーフ・装飾
- 円相（円のアウトライン）、和紙テクスチャ、罫線
- 写真使う場合は、80 年代の街角風景・レコードジャケット風・ノートのような懐かしさを意識
- イラストは控えめに、サービスの雰囲気を壊さない

### NG モチーフ
- ど派手なグラデーション、絵文字てんこ盛り、若者向け SNS テンプレ調

## カルーセル構成、推奨 7 枚

### スライド 1、フック（表紙）

**目的**、スワイプを促す入口

**コピー**、
```
昭和43年生まれの方、
集まりませんか。

1968
```

**サブ**、
```
1968年生まれ限定コミュニティ
ベータテスター募集中
```

**ビジュアル指示**、
- 背景、生成り
- 中央に大きく「1968」のロゴ
- 上部に「昭和43年生まれの方、集まりませんか。」を縦組または横組で配置
- 下部右に「→ スワイプ」のヒント
- アクセントに円相モチーフ、または和紙テクスチャ

---

### スライド 2、サービス紹介

**コピー（見出し）**、
```
1968 とは
```

**本文**、
```
1968年（昭和43年）生まれの方だけが
参加できる、小さな会員制
コミュニティです。

同い年だけが集まる安心感のなかで、
本音で語り合える場所をめざしています。
```

**ビジュアル指示**、
- 上部に「1968 とは」見出し
- 中央に本文、行間広め
- 右下に小さく 1968 ロゴ

---

### スライド 3、何が話せるか

**コピー（見出し）**、
```
こんな話題で
語り合います
```

**本文（カテゴリ列挙）**、
```
🎬 アニメ・特撮の名作
🎤 歌謡曲・あの頃の音楽
📺 テレビ・ドラマの記憶
🍬 駄菓子・給食・あの味
🎮 子供の頃の遊び
💬 死語・流行語
📚 学校生活・先生・部活
🏢 バブル・社会人デビュー
🌿 今の暮らし・健康のこと
👨‍👩‍👧 家族のはなし
💴 仕事・お金・老後の設計
🍻 オフ会・集まり

— 全12カテゴリ
```

**ビジュアル指示**、
- 12 個のタグまたはアイコンをグリッドで配置
- 絵文字は控えめに使用、入れない場合は和紙風アイコンに置き換え
- 「全12カテゴリ」を最下部に強調

---

### スライド 4、特典の柱

**コピー（見出し）**、
```
ベータテスター
3 つの特典
```

**本文**、
```
特典 1
正会員プラン 1年間 無料
（通常 月額480円）

特典 2
全 12 カテゴリ自由に利用
投稿・閲覧・DM 全機能

特典 3
創設メンバーバッジ
プロフィールに永久表示
```

**ビジュアル指示**、
- 3 つの大きな数字（1・2・3）を縦に並べる
- 各特典のタイトルを太字、説明を小さく
- 「1年間 無料」を最も大きく、紺色でハイライト

---

### スライド 5、応募条件

**コピー（見出し）**、
```

お待ちしています
```

**本文**、
```
✓ 1968年（昭和43年）生まれの方

✓ 同い年とゆっくり話したい方

✓ 新しいサービスを一緒に育てる
```

**ビジュアル指示**、
- 4 項目のチェックリスト
- 各項目に和風のチェックマーク、または点描
- 余白を多めに、優しい印象に

## 注意事項

- カルーセルの 1 枚目は、Instagram のフィード上で表示される代表画像になる、最も重要
- 文字サイズは小さくしすぎない、スマホで読める大きさに（最小 24pt 推奨）
- スライド枚数は 7〜10 枚が最適、それ以上は読まれない
- インスタグラムは画像を中央クロップで表示することがあるため、重要な情報は中央 80% 内に配置
- 公開前に必ずスマホ実機で確認、PC でデザインしたものは小さい画面で潰れることがある
